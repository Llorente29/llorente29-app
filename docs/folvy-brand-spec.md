# Folvy · ficha de marca

Dirección cerrada (30/06/2026). Fuente única para el rollout. Terracota **jubilado**.

## Símbolo — "El ciclo"
Anillo abierto = la operación girando entera (compras → cocina → venta → stock).
Punto verde en el cierre = el **margen** que capturas. Es el MRP II de ciclo cerrado hecho logo.

- `folvy-mark.svg` — solo marca (tinta + verde), fondo claro
- `folvy-logo-ciclo.svg` — lockup marca + wordmark, fondo claro
- `folvy-logo-ciclo-dark.svg` — lockup para fondo oscuro
- `folvy-favicon.svg` — versión tile para 16–32 px (no parece "cargando")

Wordmark: **Space Grotesk** 600, minúsculas, tracking −2.

## Paleta
| Rol | Hex | Uso |
|-----|-----|-----|
| Tinta | `#15171A` | texto, estructura, botón primario, chrome |
| Lienzo | `#F6F7F8` | fondo de la app (claro) |
| Tarjeta | `#FFFFFF` | superficies |
| Línea | `#E9EBED` | hairlines |
| **Margen sano** | `#1F9D6B` | el verde = gana dinero (acento principal) |
| Aprieta | `#C2890F` | margen ajustado |
| Pierde | `#E0492E` | margen negativo |

Tesis: **en Folvy, el color es dinero.** El chrome es monocromo; la única cromática fuerte la pone la salud del margen (verde/ámbar/rojo). Las fotos del plato ponen el apetito.

## Tipografía
- **Display** (titulares, nombres de plato, cifras hero): Space Grotesk 500/600
- **UI / cuerpo**: Inter 400/500
- **Números / coste**: JetBrains Mono (tabular) — el "al céntimo" como seña

## Chrome claro vs oscuro
- **Escritorio = claro** (barra lateral clara, luminoso, profesional).
- **Oscuro solo en modo cocina (tablet)**, donde el alto contraste es funcional (reflejos, mirar de lejos en barra).

## Densidades
Misma UI, dos densidades por token: **Oficina** (compacta, todo el dato) y **Cocina** (táctil, glanceable, alto contraste).
