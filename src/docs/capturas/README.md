Capturas de pantalla del manual
Esta carpeta contiene las capturas que se usan en el manual.
Cuando hagas screenshots de la app:
Guárdalas en formato PNG
Nombres descriptivos en kebab-case: `personal-listado.png`, `trabajador-home.png`, etc.
Resolución recomendada: 1280x720 o superior
Recorta lo importante (no toda la pantalla del móvil/escritorio)
Las referencias a estas capturas están en los archivos `.md` con el formato:
```markdown
![Descripción](../capturas/nombre-de-la-imagen.png)
```
